# 🤖 بوت ديسكورد متكامل

بوت ديسكورد متكامل مبني بـ Node.js + discord.js مع لوحة تحكم ويب.

## ✨ الميزات

- 🎫 **نظام تذاكر** - فتح، إغلاق، استلام تذاكر بفئات مختلفة
- ❓ **أسئلة شائعة (FAQ)** - ردود تلقائية وأوامر
- 🛡️ **مودريشن** - باند، كيك، ميوت، تحذيرات، حذف رسائل
- 🎉 **ترحيب** - رسائل ترحيب تلقائية + رتبة تلقائية
- ℹ️ **معلومات** - serverinfo، userinfo، botinfo، ping
- 🖥️ **لوحة تحكم ويب** - إحصائيات البوت والسيرفرات

## ⚙️ الإعداد

1. انسخ المشروع
2. ثبّت المكتبات: `pnpm install`
3. أنشئ ملف `.env` وضع:
   ```
   DISCORD_TOKEN=your_bot_token
   PORT=5000
   ```
4. شغّل: `pnpm --filter @workspace/api-server run dev`

## 📝 الأوامر

| الأمر | الوصف |
|-------|-------|
| `/ticket setup` | إعداد نظام التذاكر |
| `/ticket close` | إغلاق تذكرة |
| `/faq list` | عرض الأسئلة الشائعة |
| `/ban @user` | حظر عضو |
| `/kick @user` | طرد عضو |
| `/mute @user` | كتم عضو |
| `/warn @user` | تحذير عضو |
| `/clear 10` | حذف رسائل |
| `/serverinfo` | معلومات السيرفر |
| `/ping` | فحص سرعة الاستجابة |
